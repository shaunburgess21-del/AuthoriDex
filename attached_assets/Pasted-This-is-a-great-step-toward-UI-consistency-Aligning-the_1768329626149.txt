1. The "+ Suggest" Modal (Curate the Profile)
Since you already have the Celebrity Search/Dropdown and the File Upload, you want to avoid clutter. However, for a profile photo specifically, there are two high-value elements that would improve quality control without overwhelming the user:

Source URL / Attribution (Input Field):

Why: "FameDex" implies a public directory. Profile photos often have copyright considerations. Adding a simple text field for "Image Source" or "Photographer Credit" adds legitimacy to the platform and helps with moderation later.

A "Preview & Crop" Step (Visual Interaction):

Why: Users often upload rectangular images, but profile avatars are usually circular or square.

How: Instead of a new input field, this happens after the file upload. Once the file is selected, show a small preview with a circular overlay. This ensures the user knows exactly how the face will look on the leaderboard before they hit submit.

Summary of Modal Fields:

Who is this for? (Search/Dropdown - Existing)

Upload Image (File Picker - Existing)

Source/Credit (Text Input - New Recommendation)

Submit Button (Triggers the upload)
