Detailed requirements

Create a single reusable component for these labels, e.g. CategoryPill (if one already exists, extend it instead of creating a new one).

It should accept a category prop, e.g. "Tech" | "Music" | "Politics" | "Business" | "Sports" | "Creator" | ....

For the pill itself:

Maintain:

Current padding, border radius, font size and font weight.

Dark base background.

Add:

A subtle tinted background (low opacity over the dark base, like the attached example).

A 1px border in the same colour family.

The visual should feel like a “neon glass” chip, not a solid colour button:

No full, flat bright fills.

Colours should be slightly desaturated / low-opacity.

Example Tailwind-style class composition (adapt to real styles):

<span
  className={clsx(
    'inline-flex items-center rounded-full px-3 py-1 text-xs font-medium',
    'bg-surface-elevated border',
    categoryStyle.bg,
    categoryStyle.border,
    categoryStyle.text,
  )}
>
  {category}
</span>
