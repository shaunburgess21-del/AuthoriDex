Backend/data plan suggestion (keep it clean)
Instead of building a bunch of new endpoints, do one:

Return a single structured object:

signals.search: { volume, delta, relatedSearches[], peopleAlsoAsk[] }
signals.news: { count, headlines[], provider, updatedAt }
signals.wiki: { views24h, deltaVsBaseline }
signals.drivers: { items[], activeSources }
rank: { overall, category, categoryRank }

This keeps front-end simple and avoids multiple calls per page.
